# 인증 정보 저장 계획

## 목표
- 민간인증 자동입력에 사용할 기본 개인정보 입력 폼을 팝업에 추가
- 팝업에서 입력한 값을 `chrome.storage.local` 에 저장하고 다시 불러오는 흐름을 마련

## 작업 범위
- 저장 대상 필드 정의
- 입력값 기본값과 타입 정의
- 팝업 입력 폼 UI 추가
- 로컬 저장, 불러오기 로직 연결
- 최소 입력값 정리 규칙 적용

## 제외 범위
- 실제 자동입력 실행
- 콘텐츠 스크립트와 메시지 연동
- 사이트별 셀렉터 대응
- 민감정보 암호화 저장

## 저장 대상
- 이름
- 생년월일 8자리
- 휴대폰 번호

## 검증 기준
- 팝업에서 필드를 입력할 수 있음
- 저장 후 팝업을 다시 열면 기존 값이 유지됨
- `npm run build` 성공
- `npm run lint` 성공
