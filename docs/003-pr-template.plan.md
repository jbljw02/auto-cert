# PR 템플릿 추가 계획

## 목표
- 저장소에 Pull Request 템플릿을 추가
- `alloc` 저장소의 PR 템플릿 구조를 현재 저장소에 동일하게 반영

## 작업 범위
- `.github/PULL_REQUEST_TEMPLATE.md` 추가
- `alloc` 저장소 템플릿 형식 반영

## 제외 범위
- PR 템플릿 외 GitHub 설정 추가
- 이슈 템플릿 추가
- GitHub Actions 설정 변경

## 검증 기준
- `.github/PULL_REQUEST_TEMPLATE.md` 파일이 저장소에 존재함
- 새 PR 생성 시 템플릿 본문으로 사용할 수 있는 구조가 반영됨
