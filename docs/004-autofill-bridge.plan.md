# 자동입력 실행 연결 계획

## 목표
- 팝업에 저장된 인증 정보를 현재 탭에 전달하는 실행 흐름을 추가
- 공통 민간인증 팝업 입력 필드에 수동 자동입력을 실행하는 흐름을 마련

## 작업 범위
- 팝업에 자동입력 실행 버튼 추가
- 현재 활성 탭 대상으로 메시지 전송 로직 추가
- 콘텐츠 스크립트 메시지 수신 구조 추가
- 저장된 프로필을 전달받아 입력 시도 결과를 응답하는 최소 흐름 추가
- 공통 민간인증 팝업 입력 필드 selector 대응
- 실행 성공, 실패, 미지원 상태에 대한 기본 피드백 추가

## 제외 범위
- 사이트별 복잡한 예외 처리
- 고도화된 셀렉터 전략
- 민감정보 암호화 저장

## 공통 대상
- 이름: `#oacx_name`
- 생년월일: `#oacx_birth`
- 휴대폰번호 뒷자리 8자리: `#oacx_phone2`
- 휴대폰 앞자리는 고정값으로 간주

## 검증 기준
- 팝업에서 자동입력 실행 버튼을 누를 수 있음
- 현재 탭 콘텐츠 스크립트가 메시지를 수신함
- 공통 민간인증 팝업 필드에 자동입력이 시도됨
- 자동입력 결과를 팝업에서 확인할 수 있음
- `npm run build` 성공
- `npm run lint` 성공
